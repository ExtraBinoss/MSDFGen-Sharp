## Expected Output

The output is generated using **[MSDFtron](https://github.com/newemka/MSDFtron)**, which leverages the [msdfgen](https://github.com/Chlumsky/msdfgen) technique developed by Viktor Chlumsk√Ω. 

We utilize **[msdf-bmfont-xml](https://github.com/soimy/msdf-bmfont-xml)** to create the font texture atlas, resulting in a `.png` image and a `.fnt` descriptor. These assets are then imported into the **[Tixl](https://github.com/tixl3d/tixl)** motion graphics software for final rendering.
